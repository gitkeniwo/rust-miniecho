# rust-miniecho

``` sh
$miniecho -h
A lesser echo utility written in rust

Usage: miniecho [OPTIONS] <TEXT>...

Arguments:
  <TEXT>...  Required: actual inputs for miniecho to display

Options:
  -n, --omitnewline  Not Required: if set true, omit Newline character at the end
  -h, --help         Print help
  -V, --version      Print version
```
